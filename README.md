
# VLC组件库

### 一个vue的移动端组件库


## example

安装
```
npm install webpack -g
npm install webpack-dev-server -g
npm install

```

初始化

```
npm run init
```

开发环境
```
npm run dev

```

生产环境
```
npm run build

```


默认端口号

```
8080

```


### 使用

#### 安装

```
   npm install vlc
```

#### 注册组件


```
import Vue from 'vue';
import vlc from 'vlc';
Vue.use(vlc);

```
